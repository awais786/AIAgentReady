# Pilot: edx-platform

Manual end-to-end run of the kit on [edx-platform](https://github.com/openedx/edx-platform),
executed by hand before automating anything. Each of the five layers is hand-crafted, with
notes on what analysis each artifact required and how long every step took.

Produces:

- a reference example of ideal kit output,
- the real spec for what automation must produce,
- the effort baseline that prices client engagements,
- a before/after ten-question validation table.

**Status: placeholder.** Executed in **Phase 1** of the roadmap (1–2 weeks).
